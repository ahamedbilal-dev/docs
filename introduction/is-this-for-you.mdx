This course is designed to take you from zero to building real AI applications — but it's not the right fit for everyone. Before you dive in, it's worth spending two minutes figuring out whether this is the resource that will serve you best. The clearer you are on this upfront, the more focused and effective your learning will be.

## Who this course is designed for

This course works whether you've never written a single line of code or you're already comfortable in another programming language. The curriculum starts from absolute zero and builds systematically, but moves fast enough that programmers coming from JavaScript, Java, or another background won't find it tedious.

The key difference from other beginner courses: you'll learn Python using **professional tools and workflows from day one** — the same setup used at real AI companies. There are no simplified "beginner" environments you'll have to unlearn later.

<Tabs>
  <Tab title="This course IS for you">
    You're a great fit for this course if any of the following describe you:

    - **You want to build AI applications** and need Python skills to make that happen — whether that's calling LLM APIs, building data pipelines, or creating AI agents.
    - **You're a complete beginner** who has never written code before and wants a structured, practical path into Python.
    - **You're coming from another language** (JavaScript, Java, R, etc.) and want to get productive in Python for AI as quickly as possible.
    - **You learn by doing** — you'd rather write code and fix errors than spend hours reading theory before touching a keyboard.
    - **You're ready to install real tools** — VS Code, Git, virtual environments — and work the way professional developers actually work.
    - **You want a resource that stays current** — this handbook is updated regularly as tools and syntax evolve, unlike a static video or book.

    <Tip>
      Even if you've tried to learn Python before and gave up, this course is worth another shot. The workflow-first approach solves the problem that causes most beginners to quit.
    </Tip>
  </Tab>
  <Tab title="This course is NOT for you">
    This course probably isn't the right fit if:

    - **You want an exhaustive Python reference** covering every feature and edge case of the language. This course is focused and practical — we teach what you need for AI development, not an encyclopedia of everything Python can do.
    - **You're looking for casual hobby coding** without proper tooling. The course introduces professional tools early because they make everything easier. If that feels like more structure than you want, a more relaxed tutorial might suit you better.
    - **You already have strong Python skills** and are looking to go deep on a specific AI framework like PyTorch or a specific domain like computer vision. This course lays foundations; specialised advanced courses would serve you better.
    - **You want a purely theoretical understanding** of computer science. Every section here is oriented toward building real things, not academic study.

    <Warning>
      If you're looking for a slow, topic-by-topic walkthrough of all Python syntax, this course will feel too fast and too focused. That's by design — we optimise for getting you building AI applications, not covering every corner of the language.
    </Warning>
  </Tab>
</Tabs>

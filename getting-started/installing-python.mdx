Before you can write and run Python code, you need to install Python on your computer. The good news is that Python works the same way across all platforms once it's installed — the setup process is just slightly different depending on your operating system.

</Note>

## What Happens After Installation?

Once Python is installed, you'll be able to:

- Run Python from your terminal or command prompt
- Execute `.py` script files
- Use `pip`, Python's package manager, to install libraries
- Launch an interactive Python session for quick experiments

Every guide ends with a verification step so you can confirm everything is working before moving on.

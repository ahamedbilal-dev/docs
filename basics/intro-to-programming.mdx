Programming is the art of giving precise instructions to a computer. Unlike a human colleague who can fill in the gaps of an ambiguous request, a computer follows your directions literally — step by step, exactly as written. Understanding this fundamental truth is the first and most important mindset shift you'll make on your journey to becoming a programmer.

## What Is Programming?

Think of programming like writing a recipe. When you cook, you follow steps: heat the oil, add onions, stir for two minutes. Programming works the same way — except you're telling a computer what to do instead of describing a dish.

Every program you've ever used — from a web browser to a recommendation algorithm — is simply a very long, very precise set of instructions written by a human being.

## Computers Are Literal

Here's the key thing to internalize: computers cannot guess what you mean. If you ask a friend to "make coffee," they know what to do. A computer needs every single detail spelled out:

<Steps>
  <Step title="Get a cup" />
  <Step title="Add coffee grounds" />
  <Step title="Pour hot water" />
  <Step title="Wait 3 minutes" />
  <Step title="Add milk if desired" />
</Steps>

This literalness is both a frustration and a superpower. It means your programs behave predictably — the computer will never improvise or deviate from your instructions.

## What Is a Programming Language?

Computers ultimately speak in binary — ones and zeros. A programming language is a bridge that lets humans write instructions in a readable, structured form that can then be translated into machine instructions.

Python is one of hundreds of programming languages, but it's particularly well-suited for beginners and for AI/data work because its syntax reads almost like plain English.

## Basic Programming Operations

All programming, regardless of language, comes down to four fundamental operations:

<Tabs>
  <Tab title="Store Information">
    Save data so your program can remember and use it later.

    age = 25
    name = "Sarah"
    ```
  </Tab>
  <Tab title="Make Decisions">
    React differently based on conditions — the foundation of logic.

        wear_sunglasses()
    ```
  </Tab>
  <Tab title="Repeat Actions">

        do_pushup()

  <Tab title="Calculate Things">
    Perform math and combine values to produce results.
    ```python
    total = price + tax
    ```
  </Tab>
</Tabs>

## The Role of Python

Python is one of the most popular programming languages in the world — especially in data science, machine learning, and AI. Its design philosophy prioritises readability, which means Python code often looks closer to plain English than other languages. That makes it an ideal first language.

Here's a complete, runnable Python program that demonstrates several basic ideas at once:

```python
# Store information
name = "Alice"
score = 92

# Make a decision
if score >= 90:
    grade = "A"
else:
    grade = "B"

# Show the result
print(f"{name} earned a grade: {grade}")
# Output: Alice earned a grade: A
```

## Your Mindset as a Programmer

<Tip>
  Programming is a skill, not a talent. Anyone can learn it with consistent practice. When something doesn't work, that's not failure — it's debugging, and it's a core part of the job.
</Tip>

As you move through this course, keep these principles in mind:

- **Be precise.** Small typos and missing punctuation matter.
- **Break problems down.** Big tasks become manageable when split into small steps.
- **Read error messages.** They tell you exactly what went wrong and where.
- **Experiment freely.** Running code and seeing what happens is the fastest way to learn.

Right now this may feel abstract — that's completely normal. Each concept will click into place as you write more Python. Programming is a skill like any other: it takes practice, but anyone can learn it.
